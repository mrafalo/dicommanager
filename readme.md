# DICOM Manager

## Overview

Application for mass anonymization of DICOM files. It works very fast, even on thousands of files.


Dependencies:

* csv 
* dicom 
* dicom-pixeldata 
* indicatif 
* rayon 
* walkdir 
* clap 
* xlsxwriter
